# Клиентская игра Race

### Об игре

Гонки на двоих

### Информация

Игра была написана в рамках изучения курса на Udemy по Phaser.js, но была расширена уже своими силами и дальше

## Курс: https://www.udemy.com/course/phaser-3/

---

Установка приложения

```sh
npm install
```

Сборка приложения

```sh
npm run build
```

Запуск

```sh
node server.js
```
